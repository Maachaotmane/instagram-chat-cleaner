# Instagram Chat Cleaner

Automate the process of deleting Instagram direct message chats using Selenium WebDriver. This script logs into your Instagram account, scrolls through your inbox, opens each conversation, accesses the conversation info panel, and deletes chats one by one.

---

## Features

- Logs in securely using your Instagram credentials
- Scrolls through all direct message chats automatically
- ...

---

## Prerequisites

- Python 3.7+
- Google Chrome browser
- ChromeDriver compatible with your Chrome version
- Selenium package (`pip install selenium`)
